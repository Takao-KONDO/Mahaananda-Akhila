はじめに

瀬戸さん

大変ご無沙汰しております。ご機嫌いかがですか。

概況

お蔭様で、そこそこ元気に会社員しています。
ろくに活躍できていませんので低収入ですが、高校生の頃には20代半ばまで乗り切れるかも怪しかったので、既にその倍も生きているのは自分なりに上出来かと。

コロナ禍で、勤め先の業績は急落しましたが、僕の職種の業務はなくなっていませんので、当面は立場は守られると見ています。
市内の一部に行動範囲を限りつつ、ほぼ普通に生活しています。趣味の活動はだいぶ減りましたが、それでも気分転換やお遊び程度に内々で少しはやっています。

仕事

神戸市に本社のある印刷会社に、引き続き勤務しています。

  • 現在の主な職務は、Web担当（企画・制作・運営）。
  • ずっとヒラ社員です。
  • HTML / CSS / JavaScript / WordPress / PHP（……一応）。
  • 他にサブで、校正・DTP補助など。
  • ここに転職して25年余。最初はレタッチャー（アナログ製版）でした。
  • 自社関連サイト群を中心に、取引先の複数のサイトに常時関与。
  • 斜陽の印刷業ですが、当社は市内では勝ち組に属し、M&Aを再三仕掛けています。

住まいと生活

神戸市須磨区在住。独身。母と同居中。

  • 木造一戸建て4LDK。私名義の持ち家。
  • 最寄り駅まで徒歩3～4分で、特急停車駅。
  • でも買い物はどっちへ行くにも片道10分前後歩くことになる。
  • 普段の職場まで電車利用で35分。全部徒歩なら最短距離で約50分。
  • 閑静な住宅地。浜辺まで徒歩約20分。森のある公園まで徒歩4分。
  • 明石海峡大橋を臨むなら、浜辺へ出ればいい。
  • 母のいた実家と父のいた実家はそれぞれ売却済。
  • 父は2018年に他界。母も衰弱し腰が伸ばせず歩行困難で、指の関節はリウマチで不自由、字はまともに書けず料理・裁縫もできなくなってる。他の持病もいろいろ。
  • 車やバイクはなし。母は免許返納済。

健康

メニエール病の後遺症あり。運動不足。

  • 若いころの職務事情による睡眠リズムは、僕にはなかなか堪えた。
  • 左耳に常時耳鳴り。聞こえない音域があるため、左では言葉が聞き取れない。
  • メニエール自体は寛解し、今は服薬や通院はしていない。
  • 弱かった平衡感覚はさらに弱体化。
  • もともと近視と相当な弱視のあった視力もそのまま少しずつ衰え。
  • 関節可動域の狭さ、スタミナのなさ、締まりのないボディ。
  • 運動習慣は全くなし。多少歩くようにはしていますが。
  • 食生活もグダグダ。市販の惣菜と弁当頼り。
  • ただ、体重・血圧・体温などは、毎日記録するようになりました。
  • BMIだけは、まあ大体ずっと正常範囲。中肉中背ではあります。
