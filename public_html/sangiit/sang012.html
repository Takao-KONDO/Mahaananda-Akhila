462種の7音メーラ

各ヴァラヤに所属する個々のメーラ（≒旋法）を呼ぶために、次のような略号（メーラ記号）を用いる。

即ち、各ヴァラヤの代表メーラを基準として、何スヴァラがそのメーラの主音となっているかによって、各ヴァラヤの略号の後ろに次のような子音を付加するのである。

S-svara → -s

N-svara → -n

Dh-svara → -t （※註：サンスクリットでの/dh/は、単語末では/t/。）

P-svara → -p

M-svara → -m

G-svara → -k （※註：サンスクリットでの/g/は、単語末では/k/。）

R-svara → -l （※註：サンスクリットの連声の関係で、-s,と紛れないように -r,を避ける。）

従って、もし例えば、[as]と書けば長音階のことであり、[at]と書けば自然短音階のことであり、[ras]と書けば旋律的短音階（上昇形）のことであり、[rap]と書けば旋律的長音階（下降形）のことである。

このようにして、[as]に始まり[hal]に終わる簡潔な各１音節の表記で、様々なを指し示すことができる。

（※日本語話者が使うときには、区別しにくい音を区別するために、発音は長くなるが、特殊なカナ表記を使う。パソコンでご覧の方は、下の表の記号の列にある、各メーラ記号にマウスポインタを当てると、対応するカナ表記が分かる仕組みになっている。）

１オクターヴ周期で、半音単位で音程を刻む、ストレート（＝昇降が直線的）かつ純粋（＝混成しない）で、単純（＝主音以外に規定のない）な、７音で構成されるメーラは、数学的に全部で462種類である。

それらを、66種のヴァラヤごとに、記号・連番（チャクラ表記順・001～462）・移動サ表記によってここに列挙する。他に、メーラ名と所属チャクラ名のあるものは、それらをともに示す。「地位」（＝シュレーニ）の記号の意味は、すぐ下の注の通り。

なお、調を示すためには、主音（S svara）の音名に、記号・連番・メーラ名または移動サ表記のいずれかを添えればよい。調は、異名同音のものを除くと、全部で5,544種類となる。

さらに、「22シュルティ」ベースの音度名に基づくならば、462の各メーラは、それぞれ32種または64種のバリエーションに細分される。なぜなら、主音S-svaraと「パ」相当音を除く、5つまたは6つの構成音に、高低2つずつの区別が生じるのだからである。

※注：「地位」欄の記号について

・SMM：シュッダ＝マディヤマ＝メーラ（完全四度メーラ）

・PMM：プラティ＝マディヤマ＝メーラ（増四度メーラ）

・PLM：プラティローマ＝メーラ（逆行メーラ）

・AKM：アヴァナタ＝カーヤ＝メーラ（傾身メーラ）

・VAM：ヴィカラーンガ＝メーラ（欠損肢メーラ）

・VMM：ヴィナシュタ＝ムーラ＝メーラ（変壊根メーラ）
